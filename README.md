# Just Disruptive

Single-page umbrella site linking out to the ventures run under the Just Disruptive brand.

## Preview locally

Just open `index.html` in a browser, or run a static server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Stack

Plain HTML, CSS, and a tiny bit of JS. No build step, no framework.
